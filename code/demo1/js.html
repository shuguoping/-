Object
Object Object 构造函数为给定值创建一个对象包装器。如果给定值是 null 或 undefined，将会创建并返回一个空对象，否则，将返回一个与给定值对应类型的对象。 当以非构造函数形式被调用时，Object 等同于 new Object()。
Object.assign() 用于将所有可枚举属性的值从一个或多个源对象复制到目标对象。它将返回目标对象。 const target = { a: 3, b: 4 };
const source = { b: 6, c: 7 };
const returnedTarget = Object.assign(target, source);
console.log(target);
结果：{a: 3, b: 6, c: 7}
console.log(returnedTarget);
结果：{ a: 3, b: 6, c: 7 }
Object.entries() 返回一个给定对象自身可枚举属性的键值对数组，其排列与使用 for...in 循环遍历该对象时返回的顺序一致（区别在于 for-in 循环也枚举原型链中的属性） const obj = { a: '1', b: 2 };
console.log(Object.entries(obj));
结果 [ ['a', '1'], ['b', 2] ]
const obj = { 0: 'a', 1: 'b', 2: 'c' };
console.log(Object.entries(obj));
结果： [ ['0', 'a'], ['1', 'b'], ['2', 'c'] ]
Object.freeze() 可以冻结一个对象。一个被冻结的对象再也不能被修改； const obj = {
prop: a
};
Object.freeze(obj);
obj.prop = b;
console.log(obj.prop);
结果： a

Array
Array Array 对象是用于构造数组的全局对象，数组是类似于列表的高阶对象。
Array.from() 从一个类似数组或可迭代对象中创建一个新的，浅拷贝的数组实例 console.log(Array.from('shu'));
结果： Array ["s", "h, "u"]
console.log(Array.from([3, 2, 1], x => x + x));
结果：: Array [6, 4, 2]
Array.isArray() Array.isArray() 用于确定传递的值是否是一个 Array。 Array.isArray([a, b, c]);
结果：true
Array.isArray({shu: 123});
结果：false
Array.isArray("shu");
结果：false
Array.isArray(undefined);
结果： false
Array.of() 创建一个具有可变数量参数的新数组实例，而不考虑参数的数量或类型。Array.of() 和 Array 构造函数之间的区别在于处理整数参数：Array.of(7) 创建一个具有单个元素 7 的数组，而 Array(7) 创建一个长度为7的空数组 Array.of(3); 结果： [3]
Array.of(1, 2, 3); 结果： [1, 2, 3]
Array(3); 结果： [ , , ]
Array(1, 2, 3); 结果：[1, 2, 3]

Number
Number Number 对象主要用于： 如果参数无法被转换为数字，则返回 NaN。 在非构造器上下文中 (如：没有 new 操作符)，Number 能被用来执行类型转换。
toFixed() toFixed() 方法可把 Number 四舍五入为指定小数位数的数字。 var num = new Number(13.37);
document.write (num.toFixed(1))
13.4
toExponential() toExponential() 方法可把对象的值转换成指数计数法。 var num = new Number(10000);
document.write (num.toExponential(1))
1.0e+4
toPrecision()() toPrecision() 方法可在对象的值超出指定位数时将其转换为指数计数法。 var num = new Number(10000);
document.write (num.toPrecision(4))
1.000e+4

String
String 全局对象是一个用于字符串或一个字符序列的构造函数。
search() 功能：用于检索字符串中指定的子字符串，或检索与正则表达式相匹配的子字符串。
参数：regexp/substr，必须，规定子字符串或要匹配的 RegExp 对象。
返回值：原字符串中第一次匹配到目标字符串的起始位置。
 var s = 'hello world hello';
console.log(s.search('hello')); 结果：//0
console.log(s.search(/hello/g)); 结果：//0
console.log(s.search(/hello2/)); 结果：-1
toLowerCase() & toUpperCase() 功能：把字符串转换为小写/大写。
返回值：一个新的字符串。
 var s = 'shu guo';
console.log(s.toLowerCase()); //shu guo
console.log(s.toUpperCase()); //SHU GUO
concat() 功能：用于连接两个或多个字符串。
语法：stringObject.concat(stringX,stringX,...,stringX)
参数：
返回值：衔接后的一个新字符串。
 var s1 = 'hello ';
var s2 = 'world ';
var s3 = '123';
console.log(s1.concat(s2,s3)); 结果：hello world 123

min() min() 方法可返回指定的数字中带有最低值的数字。 document.write(Math.min(5,7) ) 5
document.write(Math.min(-3,5) ) -3
random() 方法可返回介于 0 ~ 1 之间的一个随机数。 document.write(Math.random())
结果： 0.0664669563108135
sqrt() sqrt() 方法可返回一个数的平方根。 var a=Math.sqrt(0); 0
var b=Math.sqrt(1); 1
var c=Math.sqrt(9); 3
var d=Math.sqrt(0.64); 0.8

Json
JSON 是一种语法，用来序列化对象、数组、数值、字符串、布尔值和 null 。它基于 JavaScript 语法，但与之不同：JavaScript不是JSON，JSON也不是JavaScript。
JSON.parse() 功能：将字符串反序列化成对象
参数：JSON字符串
返回值：对象
 var jsonString = '{"name":"ryan"}'; //JSON字符串（比如从AJAX获取字符串信息）
var obj = JSON.parse(jsonString); //将字符串反序列化成对象
console.log(obj); 结果：{ name: 'ryan' }
console.log(obj.name == 'ryan');结果：true
JSON.stringify() 功能：将一个对象解析为JSON字符串
参数：对象
返回值：JSON字符串
 var obj = {name:'ryan',age:23};
var jsonString = JSON.stringify(obj);
console.log(jsonString);结果：'{"name":"ryan","age":23}'
